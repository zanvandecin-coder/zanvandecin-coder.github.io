# Алиса



Установить Steam на Debian можно несколькими способами: через терминал с  использованием менеджера пакетов APT, с помощью официального DEB-файла с сайта Steam или через Flatpak. Рассмотрим каждый метод подробно.

## Установка через терминал (APT)

Этот способ подходит для Debian 12 и 13. Для Debian 11 используйте другие методы. 

1. **Включите компонент `contrib` в источниках APT.** Он необходим для доступа к пакету `steam-installer`. Если компонент не включён, выполните соответствующие действия (например, отредактируйте файл `/etc/apt/sources.list` или используйте графический интерфейс управления репозиториями). 

**Обновите список пакетов:**

```
bash
sudo apt update
```

**Установите пакет `steam-installer`:**

```
bash
sudo apt install steam-installer -y
```

**После установки запустите Steam** из меню приложений или через терминал командой `steam`. При первом запуске Steam загрузит среду выполнения и обновится до последней версии. 

## Установка через официальный DEB-файл

1. **Скачайте DEB-файл с официального сайта Steam:**
   - Откройте 

 в браузере.

Нажмите кнопку «Установить Steam» и загрузите файл `steam_latest.deb`. 

**Установите пакет:**

- Откройте скачанный файл через менеджер установки (например, «Установить программное обеспечение» в GNOME).
- Нажмите «Установить» и введите пароль администратора при запросе. 

1. **После завершения установки запустите Steam** из меню приложений.

## Установка через Flatpak

Если вы предпочитаете использовать Flatpak, выполните следующие шаги:

1. **Установите Flatpak и Flathub, если они ещё не установлены:**

   ```
   bash
   ```

```bash
sudo apt install flatpak
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
```

**Установите Steam через Flatpak:**

```
bash
sudo flatpak install --system flathub com.valvesoftware.Steam -y
```

**Запустите Steam:**

```
bash
```

   flatpak run com.valvesoftware.Steam
   ```

## Дополнительные рекомендации

- **Драйверы графики.** Перед использованием Steam убедитесь, что у вас установлены актуальные драйверы для вашей видеокарты (NVIDIA, AMD, Intel). 

**32-битные библиотеки.** Для 64-битных систем может потребоваться включить многоархитектурность и установить дополнительные библиотеки:

```
bash
dpkg --add-architecture i386
apt update
apt install mesa-vulkan-drivers libglx-mesa0:i386 mesa-vulkan-drivers:i386 libgl1-mesa-dri:i386
```

**Права пользователя.** Пользователь, который будет запускать Steam, должен входить в группы `video` и `audio`. Если это не так, добавьте его в эти группы командой:

```
bash
usermod -a -G video,audio имя_пользователя
```

**Протон и Steam Play.** Чтобы играть в Windows-игры, включите Steam Play с Протоном. В  настройках Steam перейдите в раздел «Совместимость», активируйте  «Включить Steam Play для всех остальных игр» и выберите последнюю версию Протона.
